# wait-time-admin
View aggregated wait time statistics by resort, lift, season, month, day, and time of day. 

Administer resorts and lifts.
## Tech Stack
### Front-end
* React
* Apollo GraphQL
* Material UI
### Back-end
* Express
* Apollo GraphQL
* MS SQL Server 2012
### Hosting Platform
* Node
* Docker
* Linux
